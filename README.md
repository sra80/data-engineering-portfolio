# data & analytics portfolio

This repository hosts selected work samples and deployment assets spanning **sql**, **python**, and **azure** data engineering. 
Content is organized by technology area to make review and navigation straightforward.

## structure
```
sql/       # database objects & migrations (add your scripts)
python/    # libraries, utilities, notebooks (add your code)
azure/     # data factory / logic apps / infra templates (add your IaC)
ops/       # ci/cd workflows and helper scripts
```
