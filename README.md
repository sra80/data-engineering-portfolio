# Data Engineering Portfolio

A curated set of projects demonstrating skills in **SQL**, **PySpark/Python**, **Microsoft Fabric / Azure**, **Power BI**, and **Logic Apps**.

> Role focus: *Senior Data Engineer / BI & Data Engineer – Azure, Fabric, Power BI, Logic Apps*

## Contents
- `sql/` – Optimized T-SQL examples, views, and patterns
- `pyspark/` – Data transformations, utilities, and notebook-ready scripts
- `azure/` – Fabric/Azure pipeline templates, Logic Apps, and IaC snippets
- `powerbi/` – DAX measures, model design notes, and report screenshots
- `docs/` – Project writeups and images

## How to Use This Repo
Each folder contains a `README.md` with:
1. **Business context** – what problem this solves
2. **Solution** – technical approach and reasoning
3. **Results** – performance improvements or quality gains
4. **Skills demonstrated** – tools and techniques

## Highlights
- **SQL**: Window functions, incremental models, dynamic pivots, SCD patterns
- **PySpark**: Range joins, date explodes, partitioning strategies
- **Azure/Fabric/Logic Apps**: Pipeline orchestration, automation, parameterization, metadata-driven loads, integration
- **Power BI**: Star schema, DAX measures, performance tuning
